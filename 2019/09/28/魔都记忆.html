Sep 28, 2019

在上海

昔日，胡适写了本书叫“四十自述·在上海”，如今我也来到了上海，虽不敢自比胡适先生，但也不妨写点什么以便他日回忆品味。（其实就是无聊写点东西打发打发时间啦）

浦东

陆家嘴

我从大连飞到上海的浦东国际机场后，本想顺路坐个磁悬浮列车的，但是立刻又被过百的票价劝退，果断乘坐地铁2号线直接去陆家嘴的环球金融中心。走出陆家嘴地铁站，我看到了大上海的天空，万里无云，是个好天气，就是有点儿热。我走到一个天桥上，一眼就看到了不远处的东方明珠，那标志性的像分子结构一样的造型，让它从周围的高楼大厦中脱颖而出，成了这条gai最靓的仔。

华宝信托

哦，对了，这次来我上海是要到一家叫华宝信托的公司里实习（打杂），他们的办公处在离东方明珠不太远的一栋高端写字楼的59楼。写字楼的一楼有一家星巴克，不过在那坐着的人大多不是为了喝咖啡的，说实话在那喝咖啡应该不是什么愉快的事情，商业氛围太重了。按照约定我在那看到了我的前辈，她也是实习生不过快离职了，所以才找了我来接手。本来我还好奇，上个电梯就能去的办公室用不着专门下来接我吧，事实证明，我还是年轻。这写字楼，电梯是有权限系统的，电梯里有个打卡的装置，用公司的通行卡扫一下，才可以点亮你要去的楼层的按钮，然后办公处的门也需要权限。（其实我只要去空中大厅的华宝前台小姐姐那拿访客卡也会可以上去的，所以前辈下来接我完全是出于好心，谢谢她）

我主要是在华宝的财务部打杂（虽然我对运营部和数字信息部的兴趣其实更大些），我去到办公室，妹纸前辈带我熟悉了一下工作的地方。不算很大的办公室，但是环境很干亮整洁，设施也很完备，我个人的话图书角倒是没怎么去，去的较多的是茶水间（蜂蜜柠檬水最棒了）。走了一圈，我们回到工位上，妹纸前辈给我发了一份实习生心得让我看看，然后开始恰饭（我来到的时候，是11点半以后了，是午饭和午休的时间），得知我也还没吃后，在我的选择下，她带我去了负一楼的罗森（负一楼很多餐厅，但我吃不起2333，一份很普通的饭要40多）。

我是8月28去的，妹纸前辈9月2离职，短短的数天，我从一个啥都不会的萌新成长为一个啥都不太会的萌新，接过了给老师们打杂的重任。我的华宝实习，开始了——

青旅

刚到上海，是8月28，打算9月1再去找短租的公寓，所以就在airbnb上先定了4天的青旅。我觉得我还是要先吐个槽。首先，这个所谓的青旅，本质上就是个多人合住的民宅，远远没有它在airbnb上的照片好看。and then，我进到房间里发现连一个带锁的柜子都没有，问房东大叔肿么肥事，大叔说——“你有什么值钱的东西（可以偷）吗。”，我一时竟无言以对，仔细想想，确实没有，但没有归没有，安全性实在太差。然后，还有一点比较难受，就是房间里不知为何总有一股难闻的气味。

尽管如此，位于即墨路的这个青旅（民宿）离公司炒鸡近，走路十多分钟就能到，呦呦切克闹，室友也都还聊得来，所以我还是在那住了下来。

说白了还是因为没钱，这里一天70左右的租金在这个地段绝对是下限了，而且我是一次给了4天的租金，不住的话显然很亏。不过在我发现这里这么不安全的时候我还是想过离开的，但就在这时从厨房飘来了一股很香的烧菜味道，我突然就觉得我也许可以留下来，毕竟会做饭的，都不是坏银！（我饿了）

虹口

霍山路

9月1号我从浦东新区的即墨路民宿搬到了虹口区的霍山路酒店。酒店的名字有几分优雅，叫索纳特大酒店，其实一点不大，就两层楼，不过外观设计倒还不错。可能是因为这附近在二战时曾是犹太人难民居住的地方，所以建筑的风格都比较西式。不过这里最大的优势是——安静。大部分时候，我喜欢安静。其次这里有一扇大窗户，这个真的真的非常重要，有阳光，有风，人才能打起精神，那种没有外窗的房间，再便宜我也不建议去，压抑。还有就是，附近吃东西的小店较多，价格也远比陆家嘴那边便宜。

虽然说这里也不是这么的完美。比如离公司稍远，坐地铁加走路需要半个小时（后来发现坐渡轮快些），房间里的空调和马桶不太好用，窗户不带纱窗总会有好光的小昆虫来拜访，室内没有晒衣服的地方（不过可以拿去楼顶的天台晒）。

总的来说除了价格偏贵（一个月3000多）以外，别的都还蛮不错的。

外滩

上海滩，在那个风起云涌的时代，是多少人梦的开始以及梦的破灭的地方。我来到上海的第二个夜晚就去了浦东那边的外滩，不过真要说难以忘怀的，定是在9月的某个雨夜，独自走在浦西的外滩上，看对岸灯火璀璨，享受雨后的凉爽。

如果能遇到神秘的魔法使，定是在这样的夜晚吧。

南京路

我来到上海以后，接连下了好几天的雨，还真是好雨知时节，但是我唯一的一双鞋子就在一天大雨的清晨被雨水浇透了，无可奈何，去南京路买了一双帆布鞋。

青浦

FES

这次我来上海，有个展览是必须看的，那就是fes（fate/grand order expo shanghai)，是在上海国家会展中心举办的fgo三周年纪念展。（fgo是一款以强大的骗氪实力为人称道的手游）

我是8月31去的昼场，来到会展中心的地铁站刚出站就看到了fes的宣传海报和路标，当然，作为一个路痴，我选择跟着其他master一起走。我去到的时候，已经排起了很长的队伍，不过由于管理得当以及场地较大，我很快也就进到了展厅。

进去的时候，每个人都可以领取一个纪念福袋，这个福袋里包括了活动指南和纪念手环，卡贴，文件夹等小玩意，哦对，还有一份面，要煮的那种2333。

领完福袋，穿过英灵长廊（和龙娘合了个影），我来到空想树下，咕哒们可以亲身体验暴打空想树的快感（其实就是对着一个测力器来上一拳，然后给你显示这一拳的伤害值），可惜现实中没有贝吉塔（龙珠里的人物），否则一定很有意思。虽然是比较有毒的活动之一，但是排队的人也着实不少，我趁着fes的欧气加持，抽了一下紫色老太氵，没想到居然两次十连就出了，壮哉我大废狗。

然后去了中华体验馆，看中国传统艺术手法创作的fgo作品，哈，不是很懂欣赏，不过感jio还蛮好看的。然后到处逛了下，就到中午了，买了一个雪糕找了个歇脚的地方慢慢地吃，等grand stage开始（啊啊，吃完雪糕以后把附送的从者插牌也扔掉了，我的saber T-T）。

虽然说在现场看grand stage是挺有气氛的，看coser们走秀，看欧皇和非球的抽卡决斗，还有大佬们的最大伤害挑战赛，不过因此错过了另外两个大活动——fgo庙会和fgo桌游体验，甚是可惜。以后如果有小伙伴们参加类似的活动，记得要优先攻略只能现场体验的活动，表演啥的回家用手机看回放就好了～对了，好像在fgo庙会遇到了大人物在玩芙芙旋转木马？（怀疑是b站的boss）

还有就是商品贩卖的时间我也因为看 gs错过了，哎，不过其实也没有什么预算买啦。

一天的活动下来，总的来说虽然多少有些遗憾的地方，但是也算是不虚此行了，你看，我的那盒fes限量版面条到现在还没吃呢，我就等着啥时候你们都把它吃了，我就放到闲鱼上拍卖！
