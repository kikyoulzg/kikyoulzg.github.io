ss折腾录

Jul 29, 2017

最近买了一个月的搬瓦工的vps（比openvpn便宜好多），打算利用ss（shadowsocks）来那啥。 主要参考了两个教程:教程1 ，教程2

流程如下：

1、ssh登录vps服务器

2、在服务器上下载ss；vi /etc/sysctl.d/local.conf（优化ss）；vi /etc/shadowsocks.json（实现多用户）；vi /etc/rc.local（开机启动ss）；iptables -I INPUT -p tcp–dport 自定义端口2 -j ACCEPT ，iptables-save（配置防火墙并开启端口）；apt-get install iptables-persistent ，service iptables-persistent save（持久化一下端口信息）

3、在pc（deepin15.4.1）上安装ss客户端，我试了两个github.com/PikachuHy/shadowsocks-client/ (代理了但是ping不上Google)，shadowshocks-qt5(ping不上Google,测试延迟的时候显示错误)

不过我发现在vps服务器上是ping得了Google的，也就是说问题出在ss上（2333，难道我只能ssh然后用w3m上Google了咩=、=）
